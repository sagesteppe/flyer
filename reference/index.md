# Package index

## Exploring the data sets

All data sets included in the package.

- [`places`](https://sagesteppe.github.io/flyer/reference/places.md) :
  Places the Western Flyer passed through on Ricketts and Steinbeck's
  journey to the Sea of Cortez.
- [`route`](https://sagesteppe.github.io/flyer/reference/route.md) : A
  stylized route of the Western Flyer on Ricketts and Steinbeck's
  journey to the Sea of Cortez.
- [`topography`](https://sagesteppe.github.io/flyer/reference/topography.md)
  : A stylized topographic data set of Baja California and adjacent
  areas.
- [`bathymetry`](https://sagesteppe.github.io/flyer/reference/bathymetry.md)
  : A stylized bathymetric data set of the Sea of Cortez and adjacent
  areas.
- [`protected`](https://sagesteppe.github.io/flyer/reference/protected.md)
  : Protected areas of Mexico
- [`landcover`](https://sagesteppe.github.io/flyer/reference/landcover.md)
  : Major landcover classes of Western Mexico
- [`land`](https://sagesteppe.github.io/flyer/reference/land.md) : Land
  and islands of Western Mexico
- [`lc_pal`](https://sagesteppe.github.io/flyer/reference/lc_pal.md) :
  Color palette for landcover data
- [`collections`](https://sagesteppe.github.io/flyer/reference/collections.md)
  : Species lists for collection sites
